# Word Frequency Plotter 📊📝

This is a Python script that analyzes the frequency of words in a text file and visualizes the top N most frequent words in a bar chart.

## Usage ℹ️

```bash
python main.py -n <number>
```

## Example

```bash
python main.py -n 10
```

## Requirements 🛠️

- Python 3.x
- Pillow (PIL)
